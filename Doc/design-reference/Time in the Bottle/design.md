# Time in the Bottle — 视觉设计方案

> 副标题：*digital labor, one keystroke at a time*
>
> 一个把团队每一次敲击键盘可视化为「沙粒」落入瓶中的实时装置。沙粒不断堆积、沉降、被埋藏，用户可以拖拽「瓶子」向下挖掘已经沉淀的历史。

---

## 1. 设计概念

- **核心隐喻**：键盘输入 = 流沙。每个按键掉落一粒带颜色的沙子，沙子受重力堆积成丘，越积越深。
- **沉积层**：当沙堆超过阈值，底部沙带会被「冻结」为一条历史记录块（band），永久封存其精确像素形态，模拟地质沉积。
- **挖掘**：拖动瓶子向下，摄像机沿世界纵深滚动，可回看被埋藏的层与其中嵌入的「化石 / 文物」。
- **基调**：克制、工程感、考古学气质。暖灰纸面 + 深色沙井，单色等宽字体贯穿。

## 4. 字体与排印

- **唯一字族**：`JetBrains Mono`（400 / 500 / 700），回退 `ui-monospace, monospace`。等宽字体强化「数据 / 仪器」气质。
- **标签**：全大写 + 宽字距（`letter-spacing` 0.1–0.22em），小字号（10–12px），弱灰色。
- **标题**：12px / 700 / `0.22em` 字距，全大写。
- **正文 / 用户名**：12–12.5px，500。
- **数字**：右对齐，700，固定最小宽度避免跳动。

| 文本 | size / weight / tracking |
|---|---|
| 主标题 | 12px / 700 / .22em / uppercase |
| 副标题 | 11.5px / 400 |
| 小节标签 | 10.5px / .18em / uppercase |
| 排行计数 | 12px / 700 |
| 底部脚注 | 10px / .1em / uppercase |

## 5. 组件细节

### Header
- 左：标题 + 副标题。
- 右：`● live` 状态——6px 青色圆点，`blink` 动画 2.4s 循环呼吸。

### 瓶子（canvas）
- 深色沙井，像素化渲染（`image-rendering: pixelated`），4px 一格的颗粒。
- 右侧贴一条 4px 自定义滚动条，滑块高度与位置由可视比例 / 摄像机深度实时驱动。
- 鼠标在 canvas 上按下拖拽 = 挖掘；光标 `grab ⇄ grabbing`。
- 当处于「浏览历史」状态时，顶部浮出琥珀色 `↑ back to surface` 胶囊按钮，点击回到实时表面。
- 下方脚注：左 `the bottle · drag down to dig`，右 `N keystrokes`。

### who's pouring（排行栏）
- 每行：身份色块（9px 圆角方）+ 用户名 + 角色标签 + 计数 + 进度条。
- 进度条宽度 = `count / max`，`width .3s ease` 平滑过渡。
- 「你」这一行有 `rgba(224,162,78,.10)` 琥珀底色高亮。

### 信息条 / 提示卡
- `N grains recorded below`：浅色信息条，提示已沉积埋藏的颗粒总数。
- 深色提示卡：`type anywhere to pour amber▮`，琥珀色闪烁光标（`blink` steps 动画）；说明拖拽可挖掘。
- `empty the bottle`：幽灵按钮（透明底 + 描边），hover 变浅灰底。

## 7. 沙粒系统（视觉呈现的技术要点）

> 此部分驱动 canvas 内的全部视觉，属于「视觉呈现」的核心。

- **网格**：宽 `W=90` 列，活跃模拟区高 `200` 行；每格渲染为 `S=4px`，视口显示 `140` 行（= 560px）。
- **重力**：逐行自底向上扫描，每格优先下落，否则随机左下 / 右下滑落（falling-sand 经典算法）；扫描方向逐帧左右交替以避免偏移。
- **颜色 = 身份**：网格值 1–4 映射到四位用户色；0 为空。
- **沉积冻结**：当沙堆表面峰值越过 `FREEZE_AT=40` 且底部 16 行（`BAND`）填充率 > 50%，将该带精确像素快照存入 `frozen[]`，活跃区整体下移、摄像机补偿，使表面平滑下降而画面不跳。
- **文物 / 化石**：内置三种像素 sprite（Ammonite 菊石、Floppy 软盘、Bulb 灯泡），可作为刚性像素（不参与流动）掉入沙中并被掩埋，挖掘时重现——为沉积层提供叙事彩蛋。
- **摄像机**：`cameraY` 为浮点世界行号；`follow` 时指数趋近表面（保留 `HEADROOM=36` 行空气），拖拽时自由移动并钳制在 `[0, maxCameraY]`。
- **渲染管线**：先把可视世界行写入一张 `90×142` 的离屏 ImageData（`Uint32` 直写像素），再 `drawImage` 放大 4× 到主 canvas，配合 `-frac*S` 实现亚像素平滑滚动。

## 8. 设计原则备忘

- **单一字族、单一强调色**：等宽字 + 琥珀，避免视觉噪音。
- **暖中性灰**：所有白 / 黑带暖调，低饱和。
- **数据即装饰**：排行、计数、颗粒数是画面的全部信息密度，不堆砌多余图标或统计。
- **像素美学**：沙井刻意保留 4px 颗粒与像素化文物，与等宽字呼应，统一「低分辨率仪器」语言。
