# Handoff: Time in the Bottle

> 一个把团队键盘输入实时可视化为「流沙落瓶」的装置式原型。
> 每次敲键掉落一粒带身份色的沙子，堆积、沉降、冻结成历史地层；拖动瓶子可向下挖掘已埋藏的层。

## 核心实现要点（给开发者的摘要）

完整数值见 `design.md`，这里给最关键的：

- **单一字族**：`JetBrains Mono`（400/500/700），等宽字贯穿全部 UI。
- **单一强调色**：琥珀 `#e0a24e`（= 「你」的身份色，也用于 live 点、光标、高亮）。
- **暖中性色板**：背景 `#e7e5df`、卡片 `#faf9f6`、主文字 `#1c1c1e`，所有白/黑均带暖灰、低饱和。
- **四位用户身份色**：You `#e0a24e` / Mina `#46b6a6` / Theo `#9c82dd` / Ren `#de6e92`，同时用于沙粒、排行条、色块。
- **瓶子**：`360×560` 深色 canvas（`#161619`），像素化渲染，4px 一格颗粒。
- **沙粒系统**：falling-sand 算法（逐行重力 + 随机斜落，扫描方向逐帧交替）；沙堆过阈值后底部沙带冻结为永久历史层；摄像机指数跟随表面，拖拽时自由滚动。

## 交互与状态

- **敲键 → 落沙**：监听键盘输入，每次按下在对应用户喷口位置注入 1 粒该用户身份色的沙子。
- **拖拽瓶子 → 挖掘**：在 canvas 上按住向下拖动，摄像机沿世界纵深滚动回看历史层；光标 `grab ⇄ grabbing`。
- **回到表面**：浏览历史时点琥珀按钮，摄像机指数趋近实时表面。
- **清空**：`empty the bottle` 重置全部沙粒与计数。
- **关键状态变量**：网格数组（活跃模拟区）、`frozen[]`（已冻结历史层快照）、`cameraY`（浮点世界行号）、`follow`（是否跟随表面）、各用户计数。

动效参数（`blink` 呼吸、进度条过渡、摄像机指数跟随系数 0.16）见 `design.md` 第 6 节。
